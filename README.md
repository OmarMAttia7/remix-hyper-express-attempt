# Remix + HyperExpress showcase

I'm attempting to get Remix to work with hyper-express, any help would be appreciated!

## Requirements

- NodeJS 20+
- NPM 10+ or pnpm 9+

## Installation

```bash
npm install
```

## Usage

```bash
npm run build && npm run start
```

or alternatively

```bash
npm run build && node server.js
```
